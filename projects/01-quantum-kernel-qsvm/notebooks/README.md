# Notebooks

- `01_qsvm_quantum_kernel.ipynb` is the main beginner notebook for Project 01.
- It is designed to run on:
  - **Aer simulator** (default)
  - **IBM Quantum hardware** (optional; requires `IBM_QUANTUM_TOKEN`)

If you run into environment issues, verify:
- Python: **3.11**
- Dependencies installed from the repo root:
  ```bash
  pip install -r requirements.txt
  ```
